auf pr0gramm

¶Annahmen

  • Relaunch war am 03.05.2014 um 14:59 (siehe Tweet zum Relaunch)

¶Methodik

  • siehe Hinweise zur Methodik
  • Zeitraum 01/2007 bis 07/2014
  • Werktage
    SELECT AVG(`up`), AVG(`down`), MAX(`up`), MAX(`down`) FROM `pr0stats` WHERE WEEKDAY(FROM_UNIXTIME(`created`)) < 5
  • Wochenende
    SELECT AVG(`up`), AVG(`down`), MAX(`up`), MAX(`down`) FROM `pr0stats` WHERE WEEKDAY(FROM_UNIXTIME(`created`)) >= 5
  • Werktage vor Relaunch
    SELECT AVG(`up`), AVG(`down`), MAX(`up`), MAX(`down`) FROM `pr0stats` WHERE WEEKDAY(FROM_UNIXTIME(`created`)) < 5 AND FROM_UNIXTIME(`created`) < '2014-07-17 14:59'
  • Werktage nach Relaunch
    SELECT AVG(`up`), AVG(`down`), MAX(`up`), MAX(`down`) FROM `pr0stats` WHERE WEEKDAY(FROM_UNIXTIME(`created`)) < 5 AND FROM_UNIXTIME(`created`) >= '2014-07-17 14:59'
  • Wochenende vor Relaunch
    SELECT AVG(`up`), AVG(`down`), MAX(`up`), MAX(`down`) FROM `pr0stats` WHERE WEEKDAY(FROM_UNIXTIME(`created`)) >= 5 AND FROM_UNIXTIME(`created`) < '2014-07-17 14:59'
  • Wochenende nach Relaunch
    SELECT AVG(`up`), AVG(`down`), MAX(`up`), MAX(`down`) FROM `pr0stats` WHERE WEEKDAY(FROM_UNIXTIME(`created`)) >= 5 AND FROM_UNIXTIME(`created`) >= '2014-07-17 14:59'
