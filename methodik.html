pr0stats - pr0gramm.com Statistiken

Motivation

Als großer Freund von faktischen Datensammlungen, sinnvollen Verknüpfungen und ästhetischen Visualisierungen habe ich das Projekt pr0stats gestartet. Ziel war die Datengewinnung von der pr0gramm Plattform und entsprechende Verarbeitung und Visualisierung der Daten und Berechnungen.

pr0stats ist als digital-kontemporäres Kunstprojekt zu verstehen.

Außerdem könnten sich das pr0stats Script und die hier gewonnen Daten für weiterführende und intensivere Untersuchungen, z.B. wissenschaftliche Studien, eignen. Das Projekt unterliegt einer Unlizenz. Wir freuen uns bei Verwendung der Daten um Verlinkung:

<a href="http://pr0stats.github.io/pr0stats/">pr0stats</a>

pr0stats steht nicht in persönlicher oder wirtschaftlicher Verbindung den Betreibern von pr0gramm oder dem Webportal pr0gramm.com. Das Projektteam von pr0stats wurde weder von pr0gramm beauftragt, noch authorisiert, die Analysen und - insbesondere - die Datenbeschaffung durchzuführen.

Methodik

Mit dem pr0stats script (github) werden zunächst Informationen zu Uploads über die pr0gramm.com API abgerufen. Diese Daten werden dann mittels SQL Queries und manueller Aufarbeitung sowie anschließender Visualisierung in wohl konsumierbare Bilder transformiert. Weitere Informationen finden sich in der README des Repositories.

  1. Abruf der Upload-Posts
  2. Abruf der Kommentar-Anzahl und Tags
  3. Speichern der Ergebnisse in Datenbank
  • id - ID des pr0gramm-Uploads (http://pr0gramm.com/new/<id>)
  • user - Uploader
  • sf_flag - safe-for-Flag (sfw, nsfw, nsfl)
  • mark - Status des Uploaders
  • promoted - ID eines beliebten Uploads
  • up - Anzahl Upvotes
  • down - Anzahl Downvotes
  • created - Zeitstempel des Eintrags
  • fullsize - Vollbild-URL
  • source - externe Quelle (optional)
  • image - relativer Pfad zum Bild/Video
  • thumb - relativer Pfad zum Thumbnail
  • tags - Tags
    Tags werden mit | verkettet und als MySQL-Datentyp TEXT gespeichert, zukünftig geplant ist, die Tags entsprechend in einer separaten Tabelle zu verwalten
  • comments - Anzahl der Kommentare
